<summary>A. Judul Riset</summary>

  ## Judul Umum
- Pengembangan Strategi Fault-Tolerant pada Dual Active Bridge Converter untuk Meningkatkan Keandalan Solid State Transformer Tegangan Menengah
- Desain dan Implementasi Operasi Tahan Gangguan Dual Active Bridge Converter dalam Arsitektur Solid State Transformer Berbasis MVDC
- Pendekatan Terpadu Fault Detection dan Fault-Tolerant Control pada Dual Active Bridge Converter untuk Aplikasi Solid State Transformer
- Strategi Operasi Terdegradasi yang Stabil pada Dual Active Bridge Converter untuk Menjamin Kontinuitas Daya Solid State Transformer
  
  ## Judul Berbasis Kontrol (Control-Oriented)
- Perancangan Fault-Tolerant Control Berbasis Modulasi Adaptif pada Dual Active Bridge Converter untuk Solid State Transformer
- Pengembangan Kontrol Adaptif Berorientasi Keandalan pada Dual Active Bridge Converter Saat Terjadi Kegagalan Saklar Daya
- Strategi Modulasi Phase Shift Tahan Gangguan untuk Mempertahankan Stabilitas Transfer Daya pada Dual Active Bridge Converter
- Desain Reliability-Oriented Control untuk Operasi Dual Active Bridge Converter dalam Kondisi Kegagalan Parsial Perangkat Daya

  ## Judul Berbasis Topologi (Hardware & Reconfiguration)
- Pengembangan Topologi Reconfigurable Dual Active Bridge Converter dengan Kemampuan Fault-Tolerant untuk Solid State Transformer
- Desain Jalur Redundansi Saklar pada Dual Active Bridge Converter untuk Meningkatkan Ketahanan Sistem Solid State Transformer
- Arsitektur Modular Dual Active Bridge Converter dengan Mekanisme Isolasi Gangguan untuk Aplikasi Solid State Transformer

  ## Judul Berbasis Reliability & Lifetime
- Analisis dan Perancangan Operasi Fault-Tolerant Dual Active Bridge Converter Berbasis Keandalan untuk Solid State Transformer
- Pendekatan Mission-Profile Based Fault-Tolerant Operation pada Dual Active Bridge Converter dalam Sistem Solid State Transformer
- Integrasi Fault-Tolerant Control dan Prediksi Umur Pakai Komponen pada Dual Active Bridge Converter untuk Solid State Transformer
- Optimasi Trade-Off Efisiensi dan Keandalan melalui Strategi Fault-Tolerant pada Dual Active Bridge Converter

  ## Rekomendasi Utama
  - Framework Fault-Tolerant Control dan Reconfiguration Strategy pada Dual Active Bridge Converter untuk Meningkatkan Keandalan Solid State Transformer Tegangan Menengah.
